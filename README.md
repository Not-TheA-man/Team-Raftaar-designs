# Team-Raftaar-designs
This repository contains detailed CAD designs of recumbent bicycle models and an experimental car model used in simulation. The project focuses on drivetrain configurations, ergonomic design modifications, aerodynamic structuring, and simulation-based modelling.
The repository includes:
a)Vayu (Rear-Wheel Drive Recumbent Bicycle)
b)Pinaka (Front-Wheel Drive Recumbent Bicycle with Modified Ergonomics)
c)Aerodynamic Fairing Design
d)XRC Car Model (Used in Hyperskill Simulation Software)

1️⃣ Vayu – Rear Wheel Drive (RWD) Recumbent Bicycle
The Vayu model is a rear-wheel drive recumbent bicycle designed with a focus on efficient power transmission and stable frame geometry.
Key Technical Features:
Rear Wheel Drive (RWD) drivetrain configuration
Optimized frame alignment for power efficiency
Ergonomic recumbent seating position
Stable structural design for performance and control

2️⃣ Pinaka – Front Wheel Drive (FWD) Recumbent Bicycle
The Pinaka model is a modified recumbent bicycle where major design changes were implemented compared to Vayu.
Major Design Modifications:
Front Wheel Drive (FWD) configuration
Changed seat position for improved ergonomics
Redesigned handle position for better control
Modified frame geometry for structural optimization
Enhanced rider posture and balance
This model represents an iterative engineering design improvement over the Vayu prototype.

3️⃣ Aerodynamic Fairing Design
The fairing model was designed to improve aerodynamic efficiency and reduce drag in recumbent vehicle motion.
Key Features:
Streamlined outer shell structure
Drag reduction focused design
Performance-oriented aerodynamic profile
Compatible with recumbent bicycle frame layout

4️⃣ XRC Model – Car Design for Simulation (Hyperskill)
The XRC model is a car CAD design developed specifically for simulation purposes using Hyperskill simulation software.
Key Highlights:
Vehicle model created for simulation testing
Used in virtual simulation environment (Hyperskill)
Structural modelling for analysis and testing
Experimental concept design approach
